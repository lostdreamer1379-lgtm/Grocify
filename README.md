## 📖 About

Managing groceries shouldn't be difficult.

**Grocify** is a smart grocery management platform that helps users organize pantry items, track expiry dates, create shopping lists, plan meals, monitor expenses, and minimize food waste through intelligent recommendations.

Whether you're a student, working professional, or family, Grocify makes everyday grocery management effortless.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/lostdreamer1379-lgtm/Grocify.git
```

### Navigate to the project

```bash
cd Grocify
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
